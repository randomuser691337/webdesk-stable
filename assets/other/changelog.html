WebDesk 0.2.0 - Nov 15, 2024

  • Improved WebComm and DeskID services
  • Added SFW mode, filters slurs/swears from text to avoid school/work detection
  • You can now use Alt+Q to quickly close a focused app/window, or Alt+M to minimize it
  • Several bug patches and UI improvements everywhere
  • Added humidity to the Weather app
  • And more! Find them all.

    • Thanks for reading, see you in 0.2.1!

WebDesk 0.1.9 - Nov 10, 2024

  • Improved WebComm and WebChat a lot, fixed up WebCall, added Contacts, and added online indicators.
  • Added Deep Sleep, uses little resources, shows the lock screen, and keeps your DeskID going. Open Controls to try it!
  • Added a sideloading button, and custom repo settings for those in Developer Mode, in the App Market
  • Several bug patches and UI improvements everywhere
  • And more! Find them all.

    • Thanks for reading, see you in 0.2.0!

WebDesk 0.1.8 - Nov 8, 2024

  • Reminder: DeskIDs can be taken if they're not in use, so keep your WebDesk open when possible.
  • Added WebComm! Combines WebDrop, WebCall, and WebChat into one app that saves DeskIDs and makes things quicker for you.
  • Patched a lot of bugs in WebCall and WebChat, as well as in other places too.
  • WebDesk protects some files if not in Developer Mode at the UI level.
  • Improved Weather and removed sunset, due to it being inaccurate.
  • Changed process for disabling Developer Mode for security.
  • Added a read-only mode to TextEdit.
  • Improved UI in apps and WebDesk.
  • Added a built-in changelog.
  • And more! Find them all.

    • Thanks for reading, see you in 0.1.9!
