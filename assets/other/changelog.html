WebDesk 0.1.8 - Nov 8, 2024

  • Reminder: DeskIDs can be taken if they're not in use, so keep your WebDesk open when possible.
  • Added WebComm! Combines WebDrop, WebCall, and WebChat into one app that saves DeskIDs and makes things quicker for you.
  • Patched a lot of bugs in WebCall and WebChat, as well as in other places too.
  • WebDesk protects some files if not in Developer Mode at the UI level.
  • Improved Weather and removed sunset, due to it being inaccurate.
  • Changed process for disabling Developer Mode for security.
  • Added a read-only mode to TextEdit.
  • Improved UI in apps and WebDesk.
  • Added a built-in changelog.
